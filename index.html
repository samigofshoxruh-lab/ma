⚙️ Первоначальная настройка
TextVault хранит все тексты в GitHub Gist — это бесплатная общая база данных. Все пользователи пишут в один Gist. Нужно 2 минуты.
Шаг 1. Создайте аккаунт на github.com (если нет).
Шаг 2. Перейдите: Settings → Developer Settings → Tokens (classic)
Нажмите Generate new token (classic), укажите название, поставьте галочку gist, нажмите Generate token. Скопируйте токен.
Шаг 3. Вставьте токен ниже → нажмите «Создать базу данных». Готово!
